Tentang Saya

Saya adalah seorang Web Developer dan UI/UX Designer berbasis di Yogyakarta, dengan passion kuat terhadap teknologi dan desain visual.

Dengan pengalaman memimpin tim produksi teknis dan proyek pengembangan perangkat lunak, saya terbiasa mengoordinasikan workflow kompleks dan menghasilkan output berkualitas tinggi. Saya percaya bahwa kode yang baik harusnya seindah desainnya.

Saat ini saya telah diterima di Universitas Gadjah Mada (UGM) untuk program S1 Matematika Murni melalui jalur SNBP 2026, di mana saya akan mendalami logika matematika dan algoritma yang memperkuat fondasi saya dalam dunia teknologi.
