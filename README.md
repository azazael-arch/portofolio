# 🌌 Antigrafity Portfolio

Selamat datang di repositori portofolio saya! Website ini adalah ruang digital untuk memamerkan proyek-proyek saya di bidang pemrograman, pembuatan film, dan matematika.

## 🚀 Live Demo
Kamu bisa mengunjungi website ini di: [https://azazael-arch.github.io/portofolio/](https://azazael-arch.github.io/portofolio/)

## 🛠️ Tech Stack
Website ini dibangun menggunakan teknologi sederhana namun powerful:
* **HTML5** - Struktur konten
* **CSS3** - Styling dan Layout
* **JavaScript** - Interaktivitas
* **GitHub Pages** - Hosting & Deployment

Di dalam portofolio ini, saya mendokumentasikan berbagai perjalanan saya, termasuk:
- **Programming:** Proyek aplikasi dan pengembangan web.
- **Cinematography:** Karya film pendek dan skrip.
- **Mathematics:** Catatan dan eksplorasi akademik saya (calon mahasiswa Matematika Murni UGM).
